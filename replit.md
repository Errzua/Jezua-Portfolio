# Jezua Palma - Portfolio Website

## Overview
A modern, responsive portfolio website built with pure HTML, CSS, and JavaScript. Features animations, particle effects, and dynamic GitHub project loading.

## Project Structure
```
/
├── index.html          # Main portfolio page
├── css/
│   └── style.css       # Custom styles
├── js/
│   └── script.js       # Interactive functionality
├── assets/
│   ├── profile.png     # Profile image
│   └── cv.pdf          # Downloadable CV
└── package.json        # Node.js dependencies (for dev server)
```

## Technologies Used
- HTML5
- CSS3 (with custom animations)
- JavaScript (vanilla)
- Bootstrap 5
- Font Awesome icons
- AOS (Animate On Scroll)
- Particles.js

## Features
- Responsive design
- Particle background effects
- Smooth scrolling navigation
- Animated skill bars
- Dynamic GitHub projects loading via API
- Downloadable CV
- Contact section with mailto link

## Development
The site uses a simple static file server (`serve`) for local development on port 5000.

## Deployment
Configured for static deployment - serves files directly from the root directory.
